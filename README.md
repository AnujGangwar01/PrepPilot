# вҡҷпёҸ Installation

## Clone Repository

```bash
git clone https://github.com/AnujGangwar01/Interview-Agent.git
```

Move inside the project

```bash
cd Interview-Agent
```

---

## Backend Setup

```bash
cd server

npm install
```

Create a `.env` file

```env
PORT=5000

MONGO_URI=Your MongoDB Atlas URI

JWT_SECRET=Your Secret Key

GEMINI_API_KEY=Your Gemini API Key
```

Run backend

```bash
npm run dev
```

---

## Frontend Setup

```bash
cd client

npm install

npm run dev
```
